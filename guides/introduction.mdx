## Who this documentation is for

This documentation is for **practice administrators** who use the OpSync admin portal to manage bookings, patients, and practice settings.

## Key concepts

| Term | Description |
|------|-------------|
| **Patient** | A person having a procedure |
| **Booking** | A patient's procedure request, tracked from creation through completion |
| **In-Practice** | Booking where admin sets the appointment time |
| **Rapid Access** | Booking where patient selects their own appointment time |
| **Financial Consent** | Document patients sign agreeing to costs, based on their health fund |
