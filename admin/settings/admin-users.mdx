Manage who has access to the OpSync admin portal.

![Admin users](/images/admin_admin_users.png)

## User list

The list shows all administrators with access to your practice:

| Column | Description |
|--------|-------------|
| **Email** | Login email address |
| **First Name** | User's first name |
| **Last Name** | User's last name |
| **Mobile** | Contact phone number |
| **Actions** | Delete user |

Use the search filters to find specific users by email, first name, or last name.

---

## Inviting a new admin

1. Click **Invite Admin User**
2. Enter their details:
   - First Name
   - Last Name
   - Email
   - Mobile Number
3. Click **Submit**

![Invite admin user form](/images/admin_admin_user_form.png)

The new user will receive an email invitation to set up their account.

---

## Removing an admin

1. Find the user in the list
2. Click the **Delete** button
3. Confirm the deletion

<Warning>Removing an admin immediately revokes their access. They will no longer be able to log in to the admin portal.</Warning>

---

## Logging in

Admins log in at [admin.opsync.com.au](https://admin.opsync.com.au) using their email and password.

![Login screen](/images/admin_login.png)

Forgot your password? Click **Forgot password?** to reset it via email.
