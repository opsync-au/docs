Configure general settings for your OpSync practice.

![Settings](/images/admin_settings.png)

## General Configuration

Basic settings for your practice.

| Setting | Description |
|---------|-------------|
| **Admin Emails** | Comma-separated list of emails that receive admin notifications |
| **Sender Email** | Email address that appears as the sender for all outgoing emails |
| **Sender Name** | Name that appears as the sender for all outgoing emails |

---

## Private Health Funds

Manage the list of health funds and their gap types.

![Private health funds](/images/admin_settings_health_funds.png)

| Column | Description |
|--------|-------------|
| **Fund Name** | Health fund name |
| **Gap Type** | Known Gap, No Gap, or Gap |

Click **Create Fund** to add a new health fund. Use the search and filters to find specific funds.

---

## Procedures

Manage medical procedures and their associated appointment types.

![Procedures](/images/admin_settings_procedures.png)

| Column | Description |
|--------|-------------|
| **Procedure** | Procedure name |
| **Attachments** | PDF documents attached to this procedure |
| **Appointment Type** | Associated appointment type and duration |

Click **+ Add Procedure** to add a new procedure.

---

## Configurable Information

Manage your practice information used throughout the application.

![Configurable information](/images/admin_settings_configurable.png)

| Section | Description |
|---------|-------------|
| **Head Specialist** | Configure the head specialist information |
| **Practice Address** | Set your practice location details |
| **Practice Information** | Additional practice details and branding |
| **Email Footer** | Customise email footer content |

Click the settings icon on each section to edit.

---

## Appointment Types

Manage appointment types and their linked procedures.

![Appointment types](/images/admin_settings_appointment_types.png)

| Column | Description |
|--------|-------------|
| **Name** | Appointment type name (e.g., "Gastroscopy Appointment") |
| **Duration** | Length of appointment in minutes |
| **Linked Procedure** | Which procedure this appointment type is for |

---

## Bank Account Details

Manage bank account details for patient payments via bank transfer.

![Bank account details](/images/admin_settings_bank_account.png)

These details are shown to patients when they choose bank transfer as their payment method.

| Field | Description |
|-------|-------------|
| **Bank Name** | Your bank's name |
| **BSB** | Bank-State-Branch number |
| **Account Number** | Your account number |
| **Reference Prefix** | Prefix added to payment references |
| **Payment Instructions** | Instructions shown to patients |

Click **Edit Details** to update your bank information.
