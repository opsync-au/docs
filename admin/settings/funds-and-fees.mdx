Configure health funds, ancillary costs, and financial consent settings. These settings are used to generate financial consent documents for patients.

![Funds and fees configuration](/images/admin_fees.png)

## Health Funds

Private health funds and their gap configuration.

![Health funds](/images/admin_fees_health_funds.png)

| Column | Description |
|--------|-------------|
| **Fund Name** | Name of the private health fund |
| **Gap Type** | No Gap, Known Gap, or Gap |

Click **Add Health Fund** to add a new fund. Click **Edit** to modify existing funds.

<Note>The gap type determines how out-of-pocket costs are calculated and displayed on financial consent documents.</Note>

---

## Ancillary Fees

Estimated out-of-pocket costs for anaesthetic and pathology services.

![Ancillary fees](/images/admin_fees_ancillary.png)

| Column | Description |
|--------|-------------|
| **Fee Type** | Anaesthetic or Pathology |
| **Billing Type** | Default or No Gap |
| **Estimated Range** | Expected cost range (e.g., $200.00 - $300.00) |

<Note>These fee ranges are shown on financial consent documents to give patients an estimate of potential out-of-pocket costs for external providers.</Note>

---

## Hospital Fees

Day surgery fees shown on self-funded financial consent documents.

![Hospital fees](/images/admin_fees_hospital.png)

| Column | Description |
|--------|-------------|
| **Location** | Hospital or day surgery name |
| **Contact** | Phone number |
| **Day Surgery Fee** | Fee amount or "Contact for quote" |

<Note>These fees are displayed in the self-funded section of financial consent documents. If no fee is set, patients are advised to contact the hospital for a quote.</Note>

## Health Fund Caps

Configure default known gap amounts and per-fund overrides.

![Health fund caps](/images/admin_fees_health_fund_caps.png)

### Default Known Gap Amount

The default out-of-pocket amount for patients with Known Gap health funds. This can be overridden per health fund using fee caps below.

### Per-Fund Overrides

Override the default known gap amount for specific health funds.

| Column | Description |
|--------|-------------|
| **Health Fund** | Fund name |
| **Applies To** | Which procedures this cap covers |
| **Max Out-of-Pocket** | Maximum amount patient pays |

<Note>Fee caps override the default known gap amount for specific health funds on financial consent documents.</Note>

Click **Add Fee Cap** to add a new override.
