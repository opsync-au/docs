Manage the specialists in your practice.

![Specialists list](/images/admin_specialists.png)

## Specialist list

Each specialist has:

| Field | Description |
|-------|-------------|
| **Order** | Display order (use arrows to reorder) |
| **Name** | Specialist's name |
| **Booking Link** | Direct link for patient bookings |

## Adding a specialist

1. Click **+ Add Specialist**
2. Enter the specialist's name
3. Enter a URL slug (used in booking links)
4. Click **Save Changes**

## Editing a specialist

1. Click **Edit** next to the specialist
2. Update the name or slug
3. Click **Save Changes**

![Edit specialist](/images/admin_specialists_form.png)

<Note>The slug is used in the booking link URL. Use lowercase letters, numbers, and hyphens only.</Note>

## Booking links

Each specialist has a unique booking link that patients can use to book directly with that doctor. Click the copy icon to copy the link.
