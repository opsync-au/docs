View all scheduled procedures across specialists and locations.

![Procedures list](/images/admin_procedures.png)

## Filtering procedures

Use the filters to find specific procedures:

- **Search** — Find by patient name
- **Specialist** — Filter by doctor
- **Location** — Filter by hospital/surgery
- **Type** — Filter by procedure type

## Procedure list

The list shows:

| Column | Description |
|--------|-------------|
| **Date** | Scheduled procedure date |
| **Arrival** | Patient arrival time |
| **Patient** | Patient name (click to view patient) |
| **Procedure** | Type of procedure |
| **Specialist** | Assigned doctor |
| **Location** | Hospital or surgery |
| **Duration** | Expected procedure length |

## Actions

- **Copy** — Copy procedure details to clipboard
- **Download CSV** — Export the filtered list

Click on a patient name to view their full record and booking details.
