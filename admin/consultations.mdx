View all scheduled consultations across specialists.

![Consultations list](/images/admin_consultations.png)

## Filtering consultations

Use the filters to find specific consultations:

- **Search** — Find by patient name
- **Specialist** — Filter by doctor

## Consultation methods

| Method | Description |
|--------|-------------|
| **Zoom** | Video call via Zoom |
| **WhatsApp** | Video call via WhatsApp |
| **FaceTime** | Video call via FaceTime |
| **In-person** | Face-to-face at the practice |

## Consultation list

The list shows:

| Column | Description |
|--------|-------------|
| **Date** | Scheduled consultation date |
| **Time** | Appointment time |
| **Patient** | Patient name |
| **Method** | How the consultation will be conducted |
| **Specialist** | Assigned doctor |
